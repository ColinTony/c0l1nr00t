## Introducción

En este post explicaré cómo he configurado mi entorno de trabajo para ser lo más eficiente posible durante mis sesiones de CTF y Bug Bounty.

## El Sistema Operativo

Elegí **Arch Linux** por su flexibilidad y porque me permite entender mejor cómo funciona mi sistema.

```bash
sudo pacman -Syu
```

## Window Manager

Uso **i3wm** para manejar mis ventanas. Es ligero y muy configurable.

### Configuración clave

1.  Atajos de teclado para terminales.
2.  Workspaces dedicados para herramientas.

## Conclusión

Tener un entorno cómodo es clave para la productividad.
